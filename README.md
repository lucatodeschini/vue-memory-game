# Vue Memory Game

A simple memory card game built with Vue.js 3.

## How to Play

1. Click on cards to flip them over
2. Find matching pairs of cards
3. Try to complete the game with as few moves as possible

## Deployment

The game is deployed and available on [GitHub Pages](https://lucatodeschini.github.io/vue-memory-game/).

## Project Structure

- `index.html` - The main HTML file containing the game
- `README.md` - This file
- `package.json` - Project configuration and dependencies

## Technologies Used

- [Vue.js 3](https://v3.vuejs.org/) - The Progressive JavaScript Framework
- HTML5 and CSS3 for the interface

## Development

To run the game locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/lucatodeschini/vue-memory-game.git
   ```

2. Open `index.html` in your browser

No build step is required for this simple project - it's designed to run directly in the browser.
